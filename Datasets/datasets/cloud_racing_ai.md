# Cloud Racing Ethical AI Dataset

**Source:** Kaggle  
**Link:** https://www.kaggle.com/code/nghihuynh/kaggle-2022-cloud-racing-ethical-ai-rising  

## Description
This dataset is related to the Kaggle 2022 Cloud Racing competition focused on ethical AI. It includes data used to evaluate AI models under ethical constraints.

## Usage
- Can be used to train and evaluate machine learning models
- Useful for studying ethical decision-making in AI systems
